# 流程控制语句

Q:以下程序会输出什么？

```
here:
for (int i = 0; i < 5; i++) {
    System.out.println("外层:" + i);
    for (int j = 0; j < 5; j++) {
        System.out.println("内层:" + j);
        if (j == 2) {
            break here;
        }
    }
}
```

控制台输出：

```
外层:0
内层:0
内层:1
内层:2
```

跳出多重循环的方式：

-   使用带有标号的的break语句
-   外层的循环条件受到内层代码控制的限制
-   使用try/catch强制跳出循环
